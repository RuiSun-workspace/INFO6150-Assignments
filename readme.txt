HTML Tags Use
- `<header>` : Website header containing title and navigation  
- `<nav>` : Navigation menu with links  
- `<section>` : Logical grouping of content  
- `<article>` : Independent content blocks 
- `<aside>` : Sidebar with announcements and audio  
- `<footer>` : Footer with contact info and copyright  

- `<table>` : Weekly club schedule  
- `<form>` : Club registration form  
- `<input>` : Text, email, password, datalist input fields  
- `<datalist>` : Provides suggestions for interests  
- `<figure>` : Contains image and caption  
- `<figcaption>` : Caption for images  

- `<a>` : Hyperlinks, telephone, and mailto  
- `<button>` : Form submission button  
- `<audio>` : University anthem playback  
- `<video>` : Campus introduction video  
- `<details>` / `<summary>` : Expandable announcement sections
